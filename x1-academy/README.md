# x1-academy

Monorepo scaffold for the X1 Academy learning platform.

## Structure

- `docs/` — product planning docs
- `apps/web/` — Next.js frontend app
- `packages/content/` — lessons, questions, and badge metadata
- `packages/ui/` — shared UI components
- `packages/sdk/` — TypeScript helpers for score, XP, signatures, and transaction builders
- `services/api/` — Fastify/Express backend
- `programs/badges/` — Anchor badge program (phase 2+)
- `infra/` — CI and Docker setup
- `scripts/` — project scripts
