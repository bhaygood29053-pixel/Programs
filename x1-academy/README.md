# X1 Academy Monorepo

This directory contains the initial scaffold for the X1 Academy platform.

## Structure

- `docs/` product and planning docs
- `apps/web/` Next.js frontend application
- `packages/` shared content, UI, and SDK packages
- `services/api/` backend API service
- `programs/` on-chain Anchor programs (phase 2+)
- `infra/` CI/CD and Docker infrastructure
- `scripts/` utility scripts for local workflows
